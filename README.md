# CAIN
CAvity Investigation Navigator: a package for analyzing cavities and tunnels in functional proteins.

CAvity Investigation Navigator (CAIN) uses alpha space to investigate the cavity of functional proteins.

To install:

cd ~

cd CAIN/

pip install .
